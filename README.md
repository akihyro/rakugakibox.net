rakugakibox.net
===============

必要なもの
----------

* Node.js v0.12.7

ローカルリポジトリ作成
----------------------

```sh
$ git clone git@github.com:akihyro/rakugakibox.net.git
$ cd rakugakibox.net
```

依存モジュール インストール
---------------------------

```sh
$ npm install
```

タスクリスト
------------

```sh
$ npm run tasks
```

お掃除
------

```sh
$ npm run clean
```

ビルド
------

```sh
$ npm run build
```
